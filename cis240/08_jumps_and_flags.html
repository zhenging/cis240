Flag States

Flags are binary values either 1 or 0 stored in a register inside the CPU. The register is called the status register since the flags represent the current status of the CPU.

Flag Function States (Name and binary value)
SF - Sign Flag Detects if sign bit is set. This is the MSB of a register. PL (Positive Level) = 0
NG (Negative Level) = 1
CF - Carry Flag Used in register math, Carry out MSB of register (during add) or Borrow into LSB of register (during substrcts) NC (No Carry) = 0
CY (Carry) = 1
OF - Overflow Flag For signed number math indicates that adding two large positive numbers set the sign bit (reversing the polarity) NV (Not Overflow) = 0
OV(Overflow) = 1
ZF - Zero Flag When set, indicates last operation caused a register to equal zero NZ (Not Zero) = 0
ZR(Zero) = 1
