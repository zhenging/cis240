06 Computer Math

Math Operations in the CPU are performed in the Arithmetic and Logic Unit (ALU). The unit contains all the logic to perform add, substract, multiply, and divide as well as AND, OR, and NOT operations(there are many others also).

  • Substraction is adding a negative number.
  • Multiplication is repeated additions.
  • Division is repeateed substraction.

Two's Compliment Numbers and Operation

  • Two's compliment is a mathematical operation for finding the negative value of any binary number. If the value is already negative, performing two's compliment will generate the positive value of the number.
  • Two's compliment is always performed on a fixed length number, in the case of a 16 bit register, all 16 bits must be included in the calculation.
  • Two step operations. First, a NOT operation(called 1's compliment), then add one to the result.

D. Multiply 7.25h * 0C.Ch shown as an unsigned 16 bit normalized result

725h * CC = 5B17C           # 1. Multiply in hex with out the decimal point
5B.17C                      # 2. With the Decimal point
0101 1011. 0001 0111 1100
 101 1011. 0001 0111 1100   # 3. Normalization, shifting leading zero out
 101 1011. 0001 1000 0      # 4. Rounding the bit 17 (LSB)
 1011 0110 0011 0000        # Final Result

E. Show the result of d) in un-normalized binary with an implied decimal point at bit 5 of a 16 bit register

    0101 1011. 0001 0111 1100
    0101 1011. 0001 1           # 1. Round the result with 5 bits of fraction
000 0101 1011. 0001 1           # 2. Shown in a 16 bit register
0000 1011 0110 0011             # Final Result

F. Multiply 101.10101 * 1011.1101 With an implied decimal at bit 3 of a 16 bit register

101.10101 * 1011.1101 = B5 * BD # 1. Convert binary to hex without decimal point
B5 * BD = 85A1                  # 2. Calculate the hex
       1000 010.1 1010 0001     # 3. With decimal point at 9^{th} bit
       1000 010.1 11            # 4. Round the result with 3 bits of fraction
000000 1000 010.1 11            # 5. Shown in a 16 bit register
0000 0010 0001 0111             # Final Result
